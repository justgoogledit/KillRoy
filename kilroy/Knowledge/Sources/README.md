# Sources

Curated primary sources you've ingested — articles, transcripts, papers, repo deep-dives.

One file per source, dated:
`YYYY-MM-DD-<slug>.md`

To populate, build an `ingest-source` skill (see `Skills/skill-creator` for how). Until then, drop notes manually using this skeleton:

```markdown
---
date: YYYY-MM-DD
source_url: <url>
source_type: article | transcript | paper | repo
tags: [...]
---

# Title

## Why this matters
<1-2 sentences>

## Key takeaways
- bullet
- bullet

## Quotes worth keeping
> "..." — author
```
